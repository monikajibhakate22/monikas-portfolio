To-Do / Appointment Manager App

A full-stack web application that helps users manage their daily appointments and tasks efficiently.

Tech Stack:
  • Frontend: HTML, CSS, JavaScript, jQuery
  • Backend: Node.js
  • Database: MongoDB
Features:
  • User registration and login with basic authentication
  • Add, view, and manage daily appointments
  • Appointments include date and time
  • All data is securely stored in MongoDB
